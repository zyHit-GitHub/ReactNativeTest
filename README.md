# Grid网格

使用方法<br>
直接import {Grid} from 'Grid';


### 代码演示
```
<Grid cols={3}>
    <View></View>
    <View></View>
    <View></View>
</Grid>

<Grid cols={3} columnGap={8} rowGap={10}>
    <View></View>
    <View></View>
    <View></View>
</Grid>

<Grid cols={3} lineWidth={StyleSheet.hairlineWidth} lineColor="#e6e6e6">
    <View></View>
    <View></View>
    <View></View>
</Grid>
```
